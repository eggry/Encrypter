# Encrypter
A mulity-file Encrypter

using TEA algorithm 

仅供交流使用，请不要将此程序用于生产环境中

# 程序功能

支持多文件加密

目前应该能保证加解密的正确性

# 目前已知问题
输出文件将变为三倍源文件大小，原因是使用了错误的类型

UI未完善，高级加密，选择性解密未完成

数据结构有待完善，需要改变文件指针的位置，加入校验位

因为TEA算法的原因，密码只能保留前16位(128bit)，也就是说，密码“12345678901234567890”，与“1234567890123456”可得到相同的解密结果
